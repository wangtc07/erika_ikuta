まいやん
かなちゃん
 
卒業おめでとう！！
 
9年間一緒に走ってこれて
幸せだったぁ。。
 
 
 
 
 
送り出した後も
じわじわと寂しさを噛み締める日々です（ ; ; ）
 
 
 
まいやんの卒業配信ライブ
見てくださった皆さま
ありがとうございました。
 
本番もリハも、まいやんを囲む尊い瞬間に何度もじーんとしました。
 
まいやんはみんなの憧れの女神だけど
それだけじゃなくて
沢山の愛情と熱意を向けてくれたその人柄に惹かれて
大好きが伝染していったんだなぁと思います。
 
 
9年間そばにいられて　沢山笑い合えて
嬉しかったです。
 
本当にありがとう...！
 
 
 
まちゅ┃_･)ｼﾞｰ♡笑
 
 
これからもずっと大好きー！！！
 
　　
 
寂しいけれど、みんなで支え合いながらがんばるぞ〜（ ; ; ）
 
来年はファンの皆さまと会えるのかしら（ ; ; ）
 
もう今年おわるね（ ; ; ）
 
大切に過ごそう（ ; ; ）
 
 
 
 
エリカ(。・∀・。)☆彡
